# orengoldenberg.com

using https://github.com/spikebrehm/isomorphic-tutorial as base

## Dev

Requirements:

* node >= 0.10.0
* ruby, gem, bundle: you're on your own, or see INSTALL-osx.sh for OS X suggestions
* grunt: `npm install -g grunt-cli`
* bower: `npm install -g bower`

```
git clone REPO
cd REPO
bundle install
npm install
bower install
grunt server
```
